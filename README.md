# Mental Health Prediction and Recommendation System

This is a machine learning-based web application that predicts a user's stress level (Low, Medium, High) based on inputs like sleep hours, screen time, mood score, activity level, and questionnaire score.

## Features
- Predicts stress/anxiety level using ML
- Provides personalized recommendations
- Simple and user-friendly interface
- Real-world healthcare-related problem

## Tech Stack
- Python
- Flask
- Pandas
- NumPy
- Scikit-learn

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   python app.py

3. Open browser:
   http://127.0.0.1:5000

## Output
- Predicts stress level: Low / Medium / High
- Shows personalized suggestions
